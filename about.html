About

This is the bench system for Coq packages. It is used to install each package on different machine configurations and Coq versions, and share the results on this website. Coq uses OPAM as a package manager. This bench system is entirely written in Ruby. All the sources, including the sources of this website, are available on GitHub under MIT license.

This bench is made to be as simple as possible, and as reproducible as possible. Still, some errors may not be properly caught yet, especially when the current Coq version is not supported by a package. The bench tries to install each package in a fresh environment, with just OCaml, OPAM and Coq. For robustness, the dependencies are not cached and everything is cleaned up after each install.

Contribute

The main contribution we expect is CPU time. If you have a powerful computer, or are patient enough, you can run the bench on your machine using coq-bench/run, and submit the results in coq-bench/database. The simplest and safest way to start the benches is to use Docker under Linux. Other platforms should work as well, as long as you can adapt the steps described in the Dockerfile.
